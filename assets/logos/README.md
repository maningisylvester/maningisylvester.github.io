# Client logos

Drop the client logo image files in this folder using these exact filenames
so they appear automatically in the "Trusted by" band on the homepage:

- `karen-plains.png`  — Karen Plains Hotel (red square logo)
- `laziz-suites.png`  — L'Aziz Suites (gold horse on black)
- `smoove.png`        — Smoove Africa (yellow drawer logo)

PNG with transparent background is ideal. Recommended max height ~200px.
If a file is missing, the band gracefully shows the client name only.
